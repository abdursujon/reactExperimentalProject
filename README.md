# Spring Boot REST API + React Frontend

Experimental full-stack project for learning and testing **different types of REST endpoints** using **Spring Boot** with a **React frontend**.

## Purpose
- Experiment with multiple REST API patterns
- Consume APIs from a real frontend
- Understand frontendâ€“backend interaction
- Practice HTTP, JSON, CORS, and state-driven UI updates

## Tech Stack
- Backend: Spring Boot (Java)
- Frontend: React + Vite
- Styling: Bootstrap
- API Style: REST (JSON over HTTP)

## Project Scope
This project is used to try:
- Simple GET endpoints
- Query params and path variables
- Scheduled/background tasks
- Different response structures
- Frontend consumption via `fetch`
